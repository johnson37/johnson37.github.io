# Rotate

## For
```python
for letter in 'Python':     # 第一个实例
   print (letter)
 
fruits = ['banana', 'apple',  'mango']
for fruit in fruits:        # 第二个实例
   print (fruit)
 
print ("Good bye!")

```
## While
```python
count = 0
while (count < 9):
   print (count)
   count = count + 1
 
print ("Good bye!")

```
