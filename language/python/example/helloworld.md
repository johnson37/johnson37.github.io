## comment
### Introduction
注释的作用是为了代码易于阅读，在代码执行过程中并没有实际意义。
### How 
注释分成单行的注释以及多行的注释。
#### 单行注释
```python
# This is one single-line comment
```
#### 多行注释
```python
'''
This is one Multi-line commnet.
And ...
'''
```
